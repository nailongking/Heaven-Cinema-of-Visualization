# Movie-Memory-Archive-of-Visualization
# 电影记忆档案馆
简介：本项目将数据与电影记忆归档，梳理呈现豆瓣电影Top250榜单，从科幻、爱情、喜剧、悬疑、动画五类中分别选取代表作深入解读，并结合数据分析各类别的核心特征与规律。随后，进行跨类型对比分析，总结榜单在题材、时间与评分等方面的整体趋势。项目旨在通过可视化系统梳理经典电影的脉络与共性，以数据叙事承载集体记忆与艺术魅力。
